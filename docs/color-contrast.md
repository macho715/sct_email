## 3. WCAG 대비율 분석

> 계산 기준: WCAG 2.1 상대휘도 공식  
> `L = 0.2126·Rlin + 0.7152·Glin + 0.0722·Blin`  
> 대비율 = `(L1 + 0.05) / (L2 + 0.05)`

## 7. 모바일 특이사항

- 모든 `input`, `textarea` 폰트 사이즈 **16px 강제** (iOS 자동 확대 방지)
- 버튼 최소 높이 **48px**, 입력칸 최소 높이 **44px** (터치 타깃)
- 768px 이하: 사이드바가 메인 콘텐츠 위에 수직 스택으로 항상 표시
- 480px 이하: 헤더 caption 숨김, title 폰트 축소
